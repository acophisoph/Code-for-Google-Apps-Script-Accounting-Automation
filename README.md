# Code-for-Google-Apps-Script-Accounting-Automation
Please do not put in any of your personal information please when using this automation (the receipts go through gemini so do not put any personal information)

#Setup
1. Create a master folder to put all your receipts inside
2. Create a new sheets in the folder
3. Put the app scipts code into the apps script

Then put in your
Google AI API Key: https://aistudio.google.com/api-keys
Root Folder ID: This is the url of the master folder you created should look like (https://drive.google.com/drive/u/1/folders/'Root_Folder_ID') You just need to put the Root_Folder_ID
Master Sheet ID: This is the url of the master sheet you created should look like (https://docs.google.com/spreadsheets/d/'Master_Sheet_ID'/edit) You just need to put the Master_Sheet_ID
